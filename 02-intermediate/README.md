# 02. Python 중급 개념 (Intermediate)

실무에서 자주 사용하는 중급 기술을 학습합니다.

## 📚 학습 내용

1. **01_decorators**: 함수/클래스 데코레이터, wraps
2. **02_generators**: yield, 제너레이터 표현식, 이터레이터
3. **03_context_managers**: with문, contextlib
4. **04_file_io**: 파일 읽기/쓰기, CSV, JSON
5. **05_error_handling**: try-except, 커스텀 예외
6. **06_modules_packages**: 모듈 구조, __init__.py, 패키지 관리
7. **07_collections**: namedtuple, Counter, defaultdict, deque
8. **08_datetime**: 날짜/시간 처리, timezone

## 🚀 실행 방법

```bash
# 각 예제 실행
python 01_decorators.py
python 02_generators.py
# ... 나머지도 동일

# 모든 예제 한 번에 실행
python run_all.py
```

## 📝 학습 팁

- 데코레이터와 제너레이터는 Python의 핵심 기능입니다
- 컨텍스트 매니저는 리소스 관리에 필수적입니다
- 예외 처리는 안정적인 프로그램을 만드는 기초입니다
