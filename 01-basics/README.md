# 01. Python 기본 문법 (Basics)

Python의 기초를 탄탄히 다지는 단계입니다.

## 📚 학습 내용

1. **01_variables_and_types**: 변수, 기본 자료형, 타입 변환
2. **02_operators**: 산술, 비교, 논리, 비트 연산자
3. **03_control_flow**: if/elif/else, match-case
4. **04_loops**: for, while, comprehensions
5. **05_functions**: 함수 정의, 인자, 반환값, lambda
6. **06_data_structures**: list, tuple, dict, set
7. **07_strings**: 문자열 조작, 포매팅, 정규표현식
8. **08_classes_basic**: 클래스, 인스턴스, 메서드, 상속

## 🚀 실행 방법

```bash
# 각 예제 실행
python 01_variables_and_types.py
python 02_operators.py
# ... 나머지도 동일

# 모든 예제 한 번에 실행
python run_all.py
```
